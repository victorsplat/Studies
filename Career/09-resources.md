## 8. Quando Usar BR vs Gringo

| Situação | Prefira | Motivo |
|----------|---------|--------|
| Aprendendo conceito novo | 🇧🇷 BR | Mais fácil de acompanhar sem se preocupar com idioma |
| Documentação oficial | 🌎 EN | Sem atalho — docs são em inglês |
| Dúvida pontual | 🇧🇷 BR | Buscar no YouTube em português |
| Aprofundamento técnico | 🌎 EN | Conteúdo mais avançado e atualizado |
| TDD / Testes | 🌎 EN | Pouco conteúdo BR de qualidade sobre isso |
| SQL / PostgreSQL | 🇧🇷 Teo Me Why + 🌎 PGExercises | BR pra aprender, PGExercises pra praticar |
| Estrutura de Dados | 🇧🇷 Fábrica de Dev + 🌎 LeetCode | BR pra entender, gringo pra praticar |
| Carreira / Mercado | 🇧🇷 BR | RinaldoDev, Fernanda Kipper — mercado BR |
