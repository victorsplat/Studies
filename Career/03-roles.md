## Mercado de JavaScript — Nota Estratégica

| Indicador | Dado |
|-----------|------|
| Vagas Node.js no Brasil (Catho) | 358 resultados |
| Vagas React/Next pleno/sênior | R$ 12.000-22.000 (DevVagas) |
| Full Stack JS Jr (Robert Half) | R$ 6.050-8.750 |
| Concorrência | Alta — JS é a stack mais popular |
| Seu nível atual em JS | React ★★★, Node ★★, Testing Library ★ |

### Decisão
- **Focar Java primeiro**: Menos concorrência na faixa Júnior, stack 100% match, diferencial mais claro
- **JS como complemento**: Depois de estabilizar como Java Jr (~6 meses), aprender Node.js aprofunda seu full stack
- **Full Stack JS pago mais?** Sim (R$ 6k-8.7k vs R$ 3.2k-5k), mas a concorrência é proporcionalmente maior e seu JS não está no mesmo nível do Java
- **Estratégia correta**: Entrar como Java Jr → depois crescer pra Full Stack Java+React → depois Node
