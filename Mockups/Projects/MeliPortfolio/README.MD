# Meli Portfolio & Resume

Portfólio full-stack desenvolvido para candidatura ao Mercado Livre.

## Tags
`#mockup` `#completed` `#nextjs` `#portfolio`

## Objetivo
Apresentar portfólio pessoal E simular o processo seletivo de TI do Mercado Livre, unindo vitrine profissional com estudo de caso funcional.

## Contexto
Projeto real de transição de carreira. Serve como peça central do portfólio do autor, demonstrando domínio full-stack com Next.js 16, MongoDB, armazenamento de imagens e autenticação.

## Stack Tecnológica
| Camada | Tecnologia |
|--------|-----------|
| Framework | Next.js 16 (Turbopack) |
| Frontend | React 18, Tailwind v4, Framer Motion |
| Banco | MongoDB Atlas |
| Storage | Vercel Blob (imagens) |
| Auth | Bearer token (API Key) |
| Estado | Zustand + TanStack Query |
| Validação | Zod |
| i18n | Custom (PT/EN/ES) |

## Funcionalidades Principais
- Portfólio com seções Hero, Sobre, Skills, Projetos, Contato
- Galeria de imagens com CMS admin (upload, delete, edição)
- Modo Explore 3D (carrossel circular com profundidade de campo)
- Formulário de vaga IT com validação de CPF, CEP, telefone
- Internacionalização (PT-BR, EN, ES)
- Tema dark/light
- Animações Lottie + Framer Motion

## Status
**Quase completo** — funcionalidades implementadas, faltando apenas testes automatizados e integração com Sentry (monitoramento).
