# Context

Contextos prontos pra usar em IAs externas (Gemini, Claude, ChatGPT) durante o período sem monitor.

## Estrutura

```
├── gemini-system-prompt.md       ← Colar no Gemini (ou qualquer IA)
└── notebooklm/
    ├── README.md                 ← Como organizar os notebooks
    ├── 01-tdd-testing.md         ← Fontes p/ Notebook "TDD"
    ├── 02-sql-postgres.md        ← Fontes p/ Notebook "SQL + PostgreSQL"
    ├── 03-estruturas-dados.md    ← Fontes p/ Notebook "ED + LeetCode"
    └── 04-java-spring.md         ← Fontes p/ Notebook "Java + Spring"
```

## Como usar

1. Abra `gemini-system-prompt.md`
2. Copie todo o conteúdo
3. Cole como instrução inicial no Gemini (pode usar no Claude/ChatGPT também)
4. A IA agora sabe quem você é e o que está estudando

## Manutenção

Atualize os arquivos sempre que sua situação mudar (novo emprego, novo projeto, novo gap).
